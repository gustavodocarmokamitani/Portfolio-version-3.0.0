Ferramentas

Estou sempre valorizando estruturas com conteúdos simples e designs minimalistas.

HTML

HTML é uma linguagem baseada em marcação, onde marcamos os elementos para definir quais informações a página vai exibir.

Bootstrap

Bootstrap é um framework para desenvolvimento de componentes de interface sites, melhorando a experiência do usuário em um site amigável e responsivo.

CSS

CSS é uma linguagem de folha de estilo composta por "camadas", criado com o propósito de estilizar as páginas HTML.

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

Figma

Figma é um editor gráfico de vetor e prototipagem de projetos de design baseado principalmente no navegador web, com ferramentas offline adicionais para aplicações desktop para GNU/Linux, macOS e Windows.
